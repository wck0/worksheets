Section 1.2 Convergence

For us, the most important feature of a sequence will be whether or not it converges, and, in the case of convergence, what it converges to. The definition of convergence should look familiar:

A sequence \(a_n\) converges to the real number \(L\) if given any \(\varepsilon > 0\) there is an integer \(N\) so that whenever \(n \geq N\text{,}\) \(|a_n - L| < \varepsilon\text{.}\) That is, if \(a_n\) is always close to \(L\) when \(n\) is big, then \(a_n\) converges to \(L\text{.}\) We write this in the usual way: \(\displaystyle\lim_{n \to \infty}a_n = L\text{.}\) Otherwise, the sequence diverges.

If \(a_n\) is a sequence and for any \(M > 0\) there is an integer \(N\) so that whenever \(n \geq N\text{,}\) \(a_n > M\text{,}\) then we say \(a_n\) diverges to infinity and write \(\displaystyle\lim_{n \to \infty}a_n = \infty\text{.}\) If \(a_n\) is a sequence and for any \(M < 0\) there is an integer \(N\) so that whenever \(n \geq N\text{,}\) \(a_n < M\text{,}\) then we say \(a_n\) diverges to negative infinity and write \(\displaystyle\lim_{n \to \infty}a_n = -\infty\text{.}\)

Example 1.2.1.
\(\displaystyle\lim_{n \to \infty} 3n-4 = \infty\) and \(\displaystyle\lim_{n \to \infty} \frac{3n-4}{2n+1} = \frac{3}{2}\text{.}\)
Activity 1.2.1.

For each of the sequences on the previous page, determine which converge and which do not. In case of convergence, determine the limit. In case of divergence, determine which diverge to infinity, which diverge to negative infinity, and which simply diverge.

Activity 1.2.2.

Compute the limit as \(n\) goes to \(\infty\) of the sequences below. Most of the usual rules for computing limits of functions apply in the context of sequences.

(a)
\(a_n = \frac{1}{n}\text{,}\) \(n \geq 1\)
(b)
\(b_n = \frac{3n-4}{1-4n}\text{,}\) \(n \geq 0\)
(c)
\(c_n = \left(\frac{3}{4}\right)^n\text{,}\) \(n \geq 0\)
(d)
\(d_n = \frac{1}{n} - \frac{1}{n-1}\text{,}\) \(n \geq 2\)
(e)
\(e_n = \sin(\pi n)\text{,}\) \(n \geq 0\)
Activity 1.2.3.

Give an example of a sequence \(a_n\) and a continuous, non-zero function \(f\) so that the sequence \(a_n\) diverges, but the sequence \(f(a_n)\) converges. Show that both \(a_n\) and \(f(a_n)\) have the stated properties.

Activity 1.2.4.

Create a sequence \(a_n\) so that \(0 \leq a_n \leq 3\) for all \(n\text{,}\) and \(a_n\) is increasing in the sense that \(a_n \leq a_{n+1}\) for all \(n\text{.}\) Show that your sequence has the required properties. Does your sequence converge?

Activity 1.2.5.

Create a sequence \(a_n\) so that \(1 \leq a_n \leq 4\) for all \(n\text{,}\) and \(a_n\) is decreasing in the sense that \(a_n \geq a_{n+1}\) for all \(n\text{.}\) Show that your sequence has the required properties. Does your sequence converge?

Activity 1.2.6.

Create a sequence \(a_n\) so that \(2 \leq a_n \leq 3\) for all \(n\text{,}\) and \(a_n\) diverges. Is \(a_n\) increasing? Decreasing?

Activity 1.2.7.

Is it true that if \(a_n \leq b_n\) for all \(n\text{,}\) and \(b_n\) is convergent, then \(a_n\) is also convergent? If so, explain why. If not, give an example to show this.

Activity 1.2.8.

Is it true that if \(a_n \geq b_n\) for all \(n\text{,}\) and \(b_n\) is divergent, then \(a_n\) is also divergent? If so, explain why. If not, give an example to show this.
