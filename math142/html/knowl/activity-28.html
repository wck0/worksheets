Activity 5.3.3.

Using a process similar to the last activity, determine all of the following antiderivatives.

(a)
\(\displaystyle\int \sin(7x)\;dx\)
(b)
\(\displaystyle\int \sec^2(2x)\;dx\)
(c)
\(\displaystyle\int \exp(10x)\;dx\)
(d)
\(\displaystyle\int x\exp(x^2)\;dx\)
(e)
\(\displaystyle\int \sqrt{3x+5}\;dx\)
(f)
\(\displaystyle\int x\sqrt{x^2+1}\;dx\)
(g)
\(\displaystyle\int \frac{x}{\sqrt{x^2+1}}\;dx\)
(h)
\(\displaystyle\int \frac{\ln(x)}{x}\;dx\)
(i)
\(\displaystyle\int \frac{1}{x(\ln(x))^2}\;dx\)
(j)
\(\displaystyle\int \frac{x}{x-50}\;dx\) (Try \(u=x-50\text{.}\))
