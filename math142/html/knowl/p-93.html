Recall that if \(F\) and \(g\) are functions, the Chain Rule for derivatives tells us how to determine the derivative of the composition function \(F(g(x))\text{:}\)

\begin{equation*} \frac{d}{dx}F(g(x)) = F'(g(x))g'(x) \end{equation*}

If \(f(x) = F'(x)\text{,}\) we can rewrite this as

\begin{equation} \frac{d}{dx}F(g(x)) = f(g(x))g'(x)\tag{5.2.1} \end{equation}
