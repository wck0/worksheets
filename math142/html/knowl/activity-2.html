Activity 1.1.2.

Determine a closed form expression for the sequences below. That is, write each as \(a_n = \textrm{something}\text{,}\) \(n \geq m\text{.}\) (You get to choose \(m\text{.}\)) There are many (many!) correct expressions.

(a)
\(2, 5, 8, 11, 14, \dots\)
(b)
\(-3, 6, -12, 24, -48, \dots\)
(c)
\(\frac{2}{3}, \frac{4}{5}, \frac{6}{7}, \frac{8}{9}, \frac{10}{11}, \dots\)
(d)
\(-1, 1, -1, 1, -1, \dots\)
(e)
\(1, 2, 1, 4, 1, 6, 1, 8, \dots\)
